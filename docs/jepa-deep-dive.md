# JEPA 深度解析

## Yann LeCun 的 AI 路线图

### 传统 LLM 的局限性
- 缺乏规划能力
- 误差无法控制
- 只能处理语言，无法理解物理世界

### JEPA 的核心思想
**不是生成式！而是预测表征！**

```
输入图像/视频 → 编码器 → 抽象表征 → 预测器 → 目标表征
```

### JEPA 发展历程

| 版本 | 时间 | 特点 |
|-----|------|------|
| I-JEPA | 2023 | 图像掩码预测 |
| V-JEPA | 2024 | 视频时序预测 |
| V-JEPA 2 | 2025 | 物理推理基准 |
| PEVA | 2026 | 16秒场景预测 |
| VL-JEPA | 2025 | 加入语言理解 |

### 关键创新

1. **自监督学习** - 不需要人工标注
2. **预测抽象表征** - 而非像素
3. **联合嵌入** - 避免表征崩溃

### 与生成式模型对比

| 特性 | 生成式 (MAE, diffusion) | JEPA |
|-----|---------------------|------|
| 输出 | 像素/图像 | 抽象表征 |
| 误差 | 累积 | 可控 |
| 计算 | 高 | 中等 |
| 物理理解 | 弱 | 强 |

### 价值导向的 JEPA (Value-Guided JEPA)
2026年最新工作：将价值函数融入 JEPA 用于动作规划

**核心思想**：
- 学习"目标条件"的价值函数
- 在 JEPA 的表征空间中进行规划
- 不需要显式的奖励函数
